Política de Privacidade

Última atualização: 18 de julho de 2026

Também disponível em: English · Español · Català

Esta Política de Privacidade descreve como o My Days ("a app") trata as informações quando você o utiliza. My Days é uma app de calendário e eventos históricos que usa a IA Gemini do Google para gerar conteúdo sobre datas selecionadas.

1. Informações que Processamos

Ao tocar em uma data, a app envia o seguinte para a API do Google Gemini:

  • A data do calendário selecionada
  • O idioma configurado no seu dispositivo

Nenhuma informação pessoal (nome, e-mail, localização, identificadores do dispositivo ou dados de conta) é coletada, armazenada ou transmitida pela app.

2. Como os Dados São Usados

A data e o idioma são enviados exclusivamente para gerar informações históricas e astronômicas sobre essa data por meio de IA. A resposta é exibida a você e não é armazenada ou registrada pela app.

3. Serviços de Terceiros

A app utiliza a API do Google Gemini para gerar conteúdo. Os dados enviados a essa API estão sujeitos à Política de Privacidade do Google. A app não utiliza nenhum outro serviço de terceiros de análise, publicidade ou rastreamento.

4. Armazenamento de Dados

My Days não mantém servidores que armazenem dados de usuários. Nenhuma conta é necessária e nenhum dado pessoal é retido após você fechar a app.

5. Notificações Locais

My Days pode, opcionalmente, enviar uma notificação diária local com um destaque sobre a data atual. Essa notificação é agendada inteiramente no seu dispositivo usando o sistema de notificações do Android: nenhum dado é enviado para nós ou para terceiros para habilitá-la, e nenhum servidor de notificações push está envolvido. Você pode ativar, desativar ou reagendar essa notificação a qualquer momento nas Configurações da app, e pode revogar a permissão de notificações nas configurações do sistema do seu dispositivo.

6. Preferências Armazenadas no seu Dispositivo

Suas preferências de notificação (se estão ativadas, o horário agendado e o tipo de conteúdo que você quer ver) são armazenadas apenas no seu dispositivo. Esses dados nunca são transmitidos para nós ou para terceiros, e são excluídos automaticamente se você desinstalar a app.

7. Privacidade de Menores

A app não é direcionada a menores de 13 anos. Não coletamos intencionalmente nenhuma informação de menores.

8. Alterações a Esta Política

Se esta política for alterada, a versão atualizada será publicada nesta mesma URL com uma data de "última atualização" revisada.
